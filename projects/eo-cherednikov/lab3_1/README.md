# Лабораторная работа №3.1 (Классификация текстов)

## Описание

В этой лабораторной работе реализуется задача многоклассовой классификации текстов с использованием модели SVM и различных ядерных функций. Работа основана на векторных представлениях документов, полученных в результате выполнения второй лабораторной работы.

## Используемые данные

Для экспериментов используются:
- Векторные представления документов из файлов `test_embeddings.tsv` и `train_embeddings.tsv`
- Размеченные корпуса текстов из директории `annotated-corpus`, содержащей обучающие и тестовые данные, разделенные по классам

## Модели и методы

### Основные модели:
- SVM с различными ядрами:
  - Линейное ядро
  - Полиномиальное ядро
  - RBF ядро
  - Сигмоидальное ядро
- Логистическая регрессия
- MLP

### Метрики оценки:
- Precision
- Recall
- F1-score
- Accuracy
- Время обучения модели

## Результаты

### Основные выводы:
- SVM с RBF ядром показал наилучшие результаты по метрике F1-score
- MLP показывает конкурентоспособные результаты при оптимальном количестве эпох

### Оптимальные параметры:
- Оптимальное количество эпох для MLP: 100
- Оптимальная размерность после PCA: 25 компонент
- Наилучшая модель: SVM с RBF ядром

## Заключение

Работа демонстрирует эффективность различных подходов к задаче многоклассовой классификации текстов. SVM с RBF ядром показал наилучшие результаты, а снижение размерности с помощью PCA позволяет ускорить обучение без значительной потери качества. Эксперименты с отбрасыванием случайных элементов векторных представлений показали устойчивость моделей к частичной потере информации.
