# Лабораторная работа №2

## 1. Описание задания
В рамках лабораторной работы была реализована векторизация текста с использованием модели Word2Vec. Векторизация проводится на основе токенов, полученных в предыдущей лабораторной работе. Для построения вектора-эмбеддинга документа используется взвешенное среднее по эмбеддингам каждого слова в предложении, а затем среднее по всем предложениям документа. В качестве весов используются значения TF-IDF для каждого слова.


## 2. Результаты работы


1. **Построение модели Word2Vec** 
    - библиотека gensim
    - длина вектора 30
    - архитектура Skip-Gram
    - сравнение косинусного расстояния между словами из близких и далеких групп
2. **Построение TF-IDF**
    - реализация через библиотеку sklearn
    - вычисление весов для всех терминов обучающей выборки
3. **Векторизация документов**
    - для каждого предложения считается взвешенное среднее для эмбеддингов всех слов в предложении. Веса - значения TF-IDF для каждого слова
    - для документа вычисляется среднее векторных представлений всех предложений
    - запись результатов векторизации тестовой выборки в tsv-файл
