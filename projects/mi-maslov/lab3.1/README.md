# Лабораторная работа №3.1: Классификация текстов

## Описание
Многоклассовая классификация текстов с использованием SVM на датасете [AG News](https://huggingface.co/datasets/ag_news).

## Выполненные задачи

### 1. Сравнение kernel functions
Эксперименты с различными ядрами SVM:

**RBF kernel:**
- precision: 0.773
- recall: 0.765
- f1_score: 0.765
- accuracy: 0.764
- training time: 4.95 s

**Poly kernel:**
- precision: 0.775
- recall: 0.762
- f1_score: 0.762
- accuracy: 0.761
- training time: 2.29 s

**Вывод:** RBF показал лучший F1-score (0.765), выбран для дальнейших экспериментов.
