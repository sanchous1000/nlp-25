# Лабораторная работа №1: Сегментация и аннотация текста

## Описание
Токенизация, стемминг и лемматизация текстов на датасете 20 Newsgroups с формированием аннотаций в формате TSV.

## Задание

1. Реализовать сегментацию текста на предложения и токенизацию на основе регулярных выражений
2. Обработать не менее 2 усложненных случаев (телефоны, email, сокращения, адреса, эмотиконы, формулы и т.д.)
3. Выполнить стемминг с использованием библиотеки 
4. Выполнить лемматизацию 
5. Проверить результаты лемматизации, привести примеры омонимии
6. Сформировать аннотацию в формате TSV: `token\tstem\tlemma`

## Выполненные задачи

### 1. Токенизация на основе регулярных выражений
Реализована обработка сложных случаев:
- **Телефонные номера:** +7-901-000-00-00, 8(918)3213412
- **Email адреса:** user@example.com
- **Сокращения:** Dr., Mr., Mrs., Sen., Rep.
- **Адреса:** Main St. Building
- **Эмотиконы:** :), :D, :(, ;)
- **Математические формулы:** a = b*c + d^2
- **Обычные слова:** стандартная токенизация

### 2. Стемминг
Используется `SnowballStemmer` из библиотеки NLTK для английского языка.

**Примеры:**
- running → run
- categories → categori
- discusses → discuss

### 3. Лемматизация
Используется `WordNetLemmatizer` из библиотеки NLTK.

**Примеры:**
- running → running (требует POS-тег для корректной работы)
- categories → category
- better → better

### 4. Структура аннотаций
Формат TSV: `token\tstem\tlemma`

**Пример:**
```tsv
The	the	the
quick	quick	quick
brown	brown	brown
fox	fox	fox

jumps	jump	jump
over	over	over
the	the	the
lazy	lazi	lazy
dog	dog	dog
```

## Использованные технологии
- nltk (SnowballStemmer, WordNetLemmatizer, sent_tokenize)
- re (регулярные выражения)
- multiprocessing (параллельная обработка)
- pathlib
