# Лабораторная работа №2

## 1. Описание задания
В рамках лабораторной работы была реализована токенизация предложений по токенам, полученным в предыдущей лабораторной работе. Векторизация проводится при помощи модели Word2vec, для построения вектора-эмбеддинга предложения используется взвешенное среднее по эмбеддингам каждого слова. В качестве весов используются значения tfidf для каждого слова.


## 2. Результаты работы

1. **Построение модели Word2Vec** 
    - библиотека gensim
    - длина вектора 30
    - сравнение косинусного расстояния между словами из близких и далеких групп
2. **Построение Tf-Idf**
    - реализация через библиотеку sklearn
3. **Векторизация предложений**
    - для каждого предложения считается взвешенное среднее для эмбеддингов всех слов в предложении. Веса - значения tf-idf для каждого слова
    -  запись в tsv-файл
    - не векторизовано всего одно предложение: из-за битого парсинга изначальных данных токены содержат html-теги, из-за этого не векторизуются tf-idf
