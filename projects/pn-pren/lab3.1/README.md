# Lab 3.1 - Классификация текстов

Классификация новостных документов с использованием SVM на векторах из lab2.

## Что сделано

- Обучены SVM-классификаторы с разными ядрами (linear, RBF, poly)
- Протестированы разные значения max_iter (100, 500, 1000)
- Реализованы метрики: precision, recall, f1, accuracy
- Найдена оптимальная модель
- Проверено влияние удаления случайных размерностей
