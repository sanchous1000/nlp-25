Директория с результатами выполнения лабораторных работ

Данная директория содержит результаты выполнения лабораторных работ студентами. Каждый каталог содержит результаты прохождения курса конкретным студентом, внутренняя структура каталогов унифицирована и выглядит следующим образом:

.
├── lab1
│   ├── source
│   │   ├── <script_name>.py
│   │   └── ... 
│   └── README.md
├── lab2
│   ├── source
│   │   ├── <script_name>.py
│   │   ├── requirements.txt
│   │   └── ... 
│   └── README.md
└── ...

Требования к структуре проекта

    Каждую лабораторную должен сопровождать README.md с отчётом о проделанной работе;
    Директория source должна содержать исходный код лабораторной работы;
    Рекомендуется включать файл requirements.txt (или аналогичный) с зависимостями проекта.

Требования к отчёту (README.md)

Отчёт по каждой лабораторной работе должен включать следующие разделы:

    Описание задания - краткое описание выполненной лабораторной работы
    Использованные технологии и инструменты - список использованных библиотек, моделей, датасетов
    Результаты работы - описание полученных результатов, метрики качества, визуализации (при необходимости)
    Выводы - анализ результатов, выявленные проблемы и способы их решения
    Инструкция по запуску - пошаговая инструкция для воспроизведения результатов
