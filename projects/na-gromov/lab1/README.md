# Лабораторная работа №1 - Сегментация и аннотация текста

## Описание

В этой лабораторной работе реализуется задача сегментации и аннотации текста. Работа включает в себя токенизацию текста, стемминг и лемматизацию слов, а также формирование аннотированных корпусов текстов в формате .tsv.

## Используемые данные

Для экспериментов используются:
- Файлы .csv с текстами (train.csv и test.csv)
- Данные содержат поля: класс, заголовок и текст документа

## Методы и подходы

### 1. Токенизация текста
Реализована сложная токенизация, включающая:
- Обработка специальных токенов (email, телефон, валюты, URL и др.)
- Замена специальных паттернов на заглушки
- Разделение текста на предложения
- Обработка эмотиконов, адресов, математических формул и т.д.

### 2. Стемминг
- Использование SnowballStemmer из библиотеки NLTK
- Приведение слов к основе

### 3. Лемматизация
- Использование WordNetLemmatizer из библиотеки NLTK
- Приведение слов к лемме
- Учет частей речи для повышения точности лемматизации

### 4. Аннотация текстов
- Формирование файлов в формате .tsv
- Каждый токен представлен в формате: token, stem, lemma
- Структура директорий соответствует классам документов

## Эксперименты

### 1. Обработка обучающего набора данных
- Обработано 120000 документ из обучающего набора
- Применена токенизация, стемминг и лемматизация
- Сохранение в директорию `assets/annotated-corpus/train`

### 2. Обработка тестового набора данных
- Обработано 7600 документ из тестового набора
- Применена токенизация, стемминг и лемматизация
- Сохранение в директорию `assets/annotated-corpus/test`

### 3. Структура директорий
- Документы группируются по классам в соответствующие поддиректории
- Каждый класс представлен отдельной директорией
- Каждый документ сохраняется в отдельный файл с идентификатором документа

## Результаты

### Основные выводы:
- Успешно реализована сложная токенизация с обработкой специальных паттернов
- Применены методы стемминга и лемматизации для нормализации текста
- Сформированы аннотированные корпусы текстов в требуемом формате
- Обработаны как обучающие, так и тестовые данные

## Заключение

Работа успешно реализует все требования задания:
- Сегментация текста на предложения
- Токенизация с обработкой специальных паттернов (email, телефоны, адреса, эмотиконы и др.)
- Стемминг и лемматизация с использованием NLTK
- Формирование аннотированных корпусов в требуемом формате .tsv
- Сохранение данных в структурированную директорию по классам

Полученные аннотированные корпуса могут быть использованы в последующих задачах обработки естественного языка, таких как классификация текстов, тематическое моделирование и другие NLP задачи.
