# Лабораторная работа №3.1: Классификация текстов

## Описание задания

Многоклассовая классификация текстов с использованием SVM на основе векторных представлений документов из лабораторной работы №2.

## Использованные технологии

- **Модель**: SVM (Support Vector Machine)
- **Библиотеки**: scikit-learn, numpy
- **Данные**: эмбеддинги из второй лабораторной (50 измерений)
- **Выборка**: случайная размером train=20000, test=5000

## Выводы

1. Оптимальное число итераций на используемой выборке: 5000. С увеличением числа итераций качество линейного SVM растёт (от 50% до 61%).

2. Лучшая ядерная функция: RBF показал значительно лучшие результаты (87.9%) по сравнению с linear (61.4%). Нелинейное преобразование позволяет лучше разделить классы в пространстве эмбеддингов.

3. Влияние размерности При удалении измерений качество падает и медленно. Сокращение с 50 до 13 измерений привело к потере ~8% accuracy.
