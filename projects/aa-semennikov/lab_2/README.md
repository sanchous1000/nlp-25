# Лабораторная работа №2

## Описание задания
В ходе выполнения лабораторной работы был реализован метод векторизации текста посредством модели Word2Vec. Для построения вектора-эмбеддинга предложения усреднение эмбеддингов каждого токена, а векторное представление документа в свою очередь было получено усреднением эмбеддингов предложений. 

## Использованные технологии и инструменты
- Модели: Word2Vec
- Библиотеки: gensim, pandas, nltk

## Результаты работы

1. **Построение модели Word2Vec** 
    - библиотека gensim, длина вектора 30, обучение в течение 30 эпох на train.csv заняло 253.03 секунд
    - в корректном соответствии косинусного расстояния между векторным представлением токенов семантической близости слов можно убедиться, запустив демонстрацию работы модели Word2Vec в файле `w2v_demo.py`
2. **Векторизация предложений**
    - для каждого предложения считается усредняются эмбеддинги всех токенов
    - результат векторизации каждого предложения записывается в формате `<doc_id_1>	<embedding_1_component_1>	<embedding_1_component_2>	<embedding_1_component_3>	...	<embedding_1_component_M>` в tsv файл
    - векторизация тестового корпуса заняла 3.89 секунд
